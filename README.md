# Design System Builder Skill

Codex skill for creating, modifying, and auditing Figma design systems for consumer mobile apps.

## Install

Copy the skill folder into your Codex skills directory:

```bash
mkdir -p ~/.codex/skills
cp -R design-system-builder ~/.codex/skills/design-system-builder
```

Then restart Codex or open a new Codex session.

## Use

Invoke the skill in Codex with:

```text
$design-system-builder
```

The skill expects a Figma link before writing to Figma. It supports lightweight product, standard product, multi-theme systems, and custom configurations.
